# Steam Link for Kodi (on ARM)
Kodi addon to download, install and launch Steam Link on ARM systems.  

## Supported operating systems
- LibreELEC (Matrix)

## Tested hardware:
- Raspberry Pi 3

### Roadmap
- LibreELEC Leia
- OSMC
- Raspbian / Raspberry Pi OS
- Raspberry Pi 4 (No X11 support under LibreELEC)
